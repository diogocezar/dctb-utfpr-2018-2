TA06 - Automatização de tarefas com Gulp

Crie um site simples a sua livre escolha e aplique as seguintes automatizações:

  • Minificação dos arquivos js;
  • Minificação dos arquivos css;
  • Watch para modificação nestes arquivos.

Para executar a automatização é necessário executar os comandos abaixo

  • npm install;
  • gulp;
